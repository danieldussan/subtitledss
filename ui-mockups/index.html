UI Improvement Proposals
4 mockups showing concrete changes to modernize the subtitledss interface. Each targets a specific UX problem.
Current UX Problems
1
Nested tab maze. Settings have 5 sub-tabs inside the Settings tab. Users click 2-3 times to reach what they need.
2
No dashboard. The main screen is just tabs. No at-a-glance view of system status, active models, or live feed.
3
No onboarding. First-time users must figure out the workflow: download model → configure audio → start capture. No guidance.
4
Flat layout. Everything is at the same visual level. No hierarchy, no grouping, no progressive disclosure.
5
No micro-interactions. Buttons, toggles, and cards lack hover/active feedback. UI feels static and unresponsive.
Proposed Solutions
1
Dashboard Overview
Replace flat tabs with a sidebar + dashboard. At-a-glance status cards, quick actions, live transcription feed, and audio meter — all on one screen.
High Impact Layout Navigation
2
Settings Reorganization
Flat sections replace nested sub-tabs. Card-based model selection, inline controls, context-aware sub-nav. No more tab hunting.
High Impact Settings Forms
3
Quick Start Onboarding
Guided 4-step wizard for first launch. Choose model, language, translation preferences, and get a summary. Under 60 seconds.
High Impact Onboarding UX
4
Micro-Interactions & Polish
Hover glow on cards, animated toggles, button ripple effects, toast slide-ins, skeleton loading, focus rings, keyboard badges.
Detail Animation Polish
Open each mockup in full width for best experience. Annotations show what's new.
